Google Sheets setup

Step 1. Create a new Google Sheet. Add these headers in row 1:
Timestamp | Name | Email | Module | Answers | Digital Signature

Step 2. In the Sheet, go to Extensions → Apps Script. Paste the following code:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    new Date().toLocaleString(),
    data.name,
    data.email,
    data.module,
    data.answers,
    data.signature
  ]);
  return ContentService.createTextOutput(JSON.stringify({status:'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}

Step 3. Click Deploy → New deployment → Web app. Set "Who has access" to Anyone. Copy the web app URL.

Step 4. Open thinkmillonboarding.html in a text editor. Find const GOOGLE_SHEET_URL = '' near the top of the script and paste your URL between the quotes. Save and reopen.
